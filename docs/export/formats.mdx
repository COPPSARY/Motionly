# Export Formats

## Current

Motionly currently exposes MP4 export from the top toolbar.

The browser records the preview in real time using the project's canvas resolution and FPS. Keep the editor tab active while exporting.

## Current Limits

- Attached audio is not included in exported MP4 files yet.
- Export depends on browser MP4 recording support.
- WebM and GIF code exists internally but is not exposed as a supported editor workflow yet.

## Planned

- Deterministic frame capture
- Stable frame pacing, duration, and audio synchronization
- Audio inclusion in MP4 output
- Visual 24, 30, and 60 FPS choices
- Resolution and aspect-ratio presets
- 720p, 1080p, 1440p, and 4K output choices
- WebM, GIF, PNG stills, and image-sequence export
