# Motionly

Motionly is an AI-native motion graphics editor. AI generates an editable project instead of a finished video, then you refine it visually—drag, scale, tune timing, scrub, and export.

Underneath, every project is a plain, readable `.motion` file. What AI makes is a starting point you can actually direct.

## What Motionly Includes

- Left navigation rail with organized panels (Media, Audio, Text, Effects, Scenes, Adjustments, Settings)
- Centered canvas preview with the project aspect ratio
- Play, pause, reset, timeline scrubber, timecode, and frame display
- Scene, canvas, and timeline selection
- Drag-to-move and corner scaling
- Visual controls for position, scale, rotation, opacity, text, color, timing, and easing
- Timeline clips with drag-and-drop from Assets panel
- Add text, delete layers, resize the timeline, and trim layer ranges
- Audio persistence in `.motion` format with synchronized preview
- Open and save `.motion` projects
- BYOK AI Chat that generates editable projects using the current project and asset list
- Direct OpenAI, Anthropic, OpenRouter, Google Gemini, Hugging Face, or custom endpoint requests with keys stored locally
- Browser-supported MP4 export with progress

## What `.motion` Is For

`.motion` is the mechanism that makes AI output editable. It is the saved source format underneath the visual editor, so projects can be reviewed, versioned, repaired, and loaded through the same parser, scene graph, and renderer whether an assistant or a person created them.

## Current Product Focus

Motionly currently focuses on the core editing loop: useful AI-assisted drafts, reliable preview, direct manipulation, clear timeline control, clean serialization, and dependable export. AI is optional; visual editing remains the primary workflow.
