# Text Presets

Text motion should preserve readability first.

Good text reveal defaults:

- Word stagger: `80ms` to `120ms`
- Character stagger: `30ms` to `50ms`
- Duration: around `1s` to `1.2s`
- Easing: `power3.out`
- Blur: subtle, usually under `10px`

Use word reveals for important copy. Use character reveals sparingly for short hero moments or typewriter-style effects.

Avoid over-animating body text, labels, and dense UI copy.
