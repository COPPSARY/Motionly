# Motion Language Reference

`.motion` is Motionly's source format. The editor should create and update it visually, but agents and contributors need to keep the syntax clean.

## Canvas

```motion
canvas {
  size 1920x1080
  fps 60
  duration 5s
  background #020308
}
```

## Text

```motion
text title {
  value "Motion graphics"
  center
  x 0
  y 0
  size 72
  color #ffffff
  opacity 1
}
```

Use `size`, not `fontSize`.

## Asset

```motion
import "./assets/logo.svg" as logo

logo {
  center
  width 240
  opacity 1
}
```

## Animation

```motion
animate title {
  from {
    opacity 0
    y 80
    blur 10
  }

  to {
    opacity 1
    y 0
    blur 0
  }

  duration 1.2s
  delay 0s
  easing power3.out
}
```

Use `easing`, not `ease`.

## Useful Properties

- `x`
- `y`
- `width`
- `height`
- `scale`
- `rotation`
- `opacity`
- `blur`
- `brightness`
- `shadow`
- `size`
- `color`
- `center`
- `layer`

## Audio

```motion
audio "/assets/my-project/background.mp3"
```

Audio files persist in `.motion` format and play during preview.

## Timeline Clips

```motion
import "/assets/my-project/video.mp4" as bgVideo

clip bgVideo {
  track 1
  start 0s
  duration 5s
  trimIn 0s
  trimOut 0s
  volume 1
  mute false
}
```

Clips reference imported assets and appear on the timeline. They can be images, SVGs, or videos. Drag assets from the Assets panel onto the timeline to create clips visually.

Properties:
- `track`: timeline track number (default 1)
- `start`: when clip starts on timeline
- `duration`: how long clip plays
- `trimIn`: source media start offset (default 0s)
- `trimOut`: source media end offset (default 0s)
- `volume`: audio volume 0-1 (optional, default 1)
- `mute`: whether to mute clip audio (optional, default false)

## Easing

Preferred:

- `power3.out`
- `smooth`
- `soft`
- `ease-out`
- `linear`

## Product Rule

Do not make `.motion` authoring the main user workflow. Keep it readable because it is the source of truth, but build UI controls first.
