# Installation

## Quickest Start (no clone)

Run the built editor instantly with npx:

```bash
npx motionly
```

This serves the editor at `http://localhost:4173` and opens your browser. Use `--port <n>` to pick a port or `--no-open` to skip launching the browser.

## Requirements

- Node.js `20.19.0` or newer
- npm
- A modern Chromium-based browser for best MP4 export support

## Install From Source

```bash
git clone https://github.com/COPPSARY/Motionly.git
cd Motionly
npm install
```

## Run

```bash
npm run dev
```

Open the local Vite URL in your browser.

## Build

```bash
npm run build
```

## Serve The Production Build

```bash
npm run serve
```

This runs the same launcher `npx motionly` uses against your local `dist/`.

## Test

```bash
npm run test:run
```
